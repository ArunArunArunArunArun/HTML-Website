What is the Internet and the Web?

The Internet is a network which connects computers everywhere. Data travels over these networks.

The Web (WWW) is a collection of connected webpages accessible through the internet.



Web browsers such as google chrome are programs used to display web pages found on the world wide web.
The web pages are transmitted through a modem and a WiFi router.
Web pages and web servers have IP adresses. The domain name system (DNS) is a way of naming each website.
Almost all web pages are made using Hypertext Markup Language (HTML). HTML is the structure and content of the web page. Cascading Style Sheets (CSS) is the style and look of the site.
